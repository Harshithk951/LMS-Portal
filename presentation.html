The Invisible Wall

Software is built for the average user. But the average user doesn't exist.

Over 1 billion people globally experience some form of disability. Traditional digital learning platforms force users with visual, auditory, cognitive, or motor differences to adapt to poorly designed UI.

15%

of the global population has a disability, yet WebAIM found that 96% of top websites fail basic WCAG accessibility criteria.

What We Are Building

Universal Access

Instead of one rigid interface, we provide an adaptive, fluid experience containing 8 distinct accessibility themes and dynamic preferences.

AI-Powered Inclusion

We leverage generative AI pipelines to automatically process static content, converting videos and PDFs into sign language, captions, and dyslexia-friendly text.

The Tech Architecture

A modern, highly-performant React 18 single page application powered by Vite, emphasizing near-instant rendering and robust DOM manipulation for accessibility APIs.

React 18 & Typescript

Type-safe component system mapping massive state objects across the UI context.

React Router v7

Seamless SPA client-side routing optimized for screen reader focus-management.

Vanilla CSS Engine

Using raw CSS custom properties to calculate themes at browser runtime (0ms latency).

Web Speech API

Native browser speech recognition allowing 100% hands-free voice-navigation.

Framer Motion

Orchestrating micro-interactions, deeply integrated with the `prefers-reduced-motion` hook.

Lucide React

An intricate mapping of 50+ SVG icons to ensure a pristine monochrome aesthetic.

Core Features (A to Z)

  • AI Accessibility Generator: Upload files, simulate Whisper+Gemini transcription processing.
  • Course Player UI: Built-in controls, dynamic captions, transcript viewer, and sign language overlay.
  • Gamification Loop: Auto-calculating XP levels, daily streaks, automated badges.
  • Keyboard Navigation: Engineered `Space` to play/pause, `C` for captions, `T` for transcripts, full tab-index compliance.
  • Leaderboard Ranking: High-score tracking podium and metric dashboard.
  • Monochrome Typography: Stunning, distraction-free aesthetic with Lucide icons (Zero Emojis).
  • Screen Reader Mode: A specialized toggle that visually declutters the entire DOM.
  • Voice-Controlled Router: Say "Dashboard" to instantly switch views.

Innovation: The CSS Engine

Rather than shipping rigid CSS files, we map hex values to CSS custom properties dynamically inside `ThemeContext.tsx`, injecting them straight into ``.


[data-theme='high-contrast-dark'] {
    --bg-primary: #000000;
    --text-primary: #ffffff;
    --border-color: #ffffff;
    /* Yellow text for low-vision readers */
    --accent-color: #ffff00; 
}
